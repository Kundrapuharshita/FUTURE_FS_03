# FUTURE_FS_03
Cake House is a professional bakery website created to improve the bakery’s online presence and customer engagement. The website includes menu items with prices, customer reviews, contact details, and a responsive design using HTML, CSS, and JavaScript.

📌 Project Overview

Cake House is a professional bakery website created for a local bakery business to improve its online presence and attract more customers.

The website is designed with a modern responsive layout and includes bakery products, prices, customer reviews, and contact details.

✨ Features

* Attractive Homepage
* Responsive Design
* Bakery Menu with Prices
* Customer Reviews
* Contact Information
* Smooth Scrolling Navigation
* Professional UI Design

💻 Technologies Used

* HTML
* CSS
* JavaScript

🎯 Project Pitch

This website helps the bakery improve customer engagement and business visibility online.

Customers can easily view bakery products, prices, and contact information through a professional digital platform.

The responsive and attractive design improves user experience and supports business growth in the digital market.
